# Changes from Incremental Mass Rewritten (v0.7.1.6)

## v -plus.2

-   Various string improvements and standardisations
-   Removed the 'reset without Supernova' button. Supernovas can now be reset for at any time on the right icon
-   Fermion negative effects are highlighted in red to make it clear they're a debuff
-   Increased Theorem Inventory height
-   Chance to have multiple dots on Theorems increased (10% -> 20%)
-   Chance for dots to spawn on a Theorem when levelling increased (10% -> 15%)
-   Break Infinity upgrade increases chance for dots to spawn on a Theorem (+10%)
-   Protoversal Theorem's "cheaper cosmic strings" dot effect is boosted (~33%+)
-   Protoversal Theorem's "entropy scaling" dot effect is boosted (~33%+)
-   Einsten Theorem's "glyphic mass" dot effect is boosted (~33%+)
-   Parallel Extruder scaling decreased (^1.2 -> ^1.17)

## v -plus.1

-   Several resource pluralisations
-   Various other string improvements
-   The 'offline time' popup only shows if you have been away for more than 5 minutes
-   FSS Button is revealed as soon as FSS can be reset for
-   Challenge 16 button is revealed as soon as unlocked
-   Self-Infinity is cheaper (100 -> 50 points)
-   Moved starting with beyond-ranks automation from Final Star Automation (T3) to Tree Automation (T2)
-   Infinity Upgrades layout adjusted to take up more space
-   Added a Theorems count display under the Infinity Upgrades Infinity Points display
-   Added a Quarks display on the Tier 1 Normal Elemental Layer
-   Added a Dark Shadows display on the Tier 2 Normal Elemental Layer
-   Added an Exotic Atoms display on the Muonic Elemental Layer
-   Infinity Points gain is boosted (Mass / 2 -> / 1.9)
-   First corruption charger is cheaper (3 Shards -> 1 Shard)
-   Challenge 7 start is lower (1.5e76 -> 1e74 Mass)
-   Atom Upgrade 2x4 is cheaper (2e44 -> 5e42 Atoms)
-   Indium's effect is boosted (^1.1 -> ^1.15)
-   NT-bh2's effect is boosted (^1.15 -> ^1.2)
